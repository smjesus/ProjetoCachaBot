CachaBOT

Objetivos e Funções do CachaBot

O CachaBot tem por objetivo automatizar as seguintes tarefas:

  • Definir um "Cargo Provisório" para os usuarios ao entrarem no servidor.

  • Fazer a Verificação dos novos usuarios
    No canal de "Boas Vindas" que será definido pelo administrador do Servidor, sera apresentado uma mensagem de boas vindas e um botao para o usuario confirmar sua participacao na comunidade.
    Apos a confirmacao do usuario, o CachaBOT retira o cargo provisorio e atribui o "Cargo Definitivo".
    Estes cargos serao definidos pelo administrador atraves dos comandos de barra.

  • Definir a Mensagem de Boas Vindas e definir o canal de apresentacao da mensagem
    Atraves dos comandos de barra, o administrador podera definir a mensagem de boas vindas, especificar o canal onde ela sera apresentada (ver comandos de barra na proxima seção).
 

Procedimentos Iniciais para ativar o CachaBot


O primeiro passo é instalar o CachaBOT no seu servidor, clicando para isso no botão "Acidionar ao Servidor" que está na página principal.
No segundo passo deve-se atribuir ao CachaBot um cargo superior aos cargos "provisório" e o "definitivo", pois caso contrario o Bot não conseguirá atribuir os cargos aos usuários.
Por fim, o terceiro passo seria configurar as devidas permissões nos canais onde o CachaBot irá atuar (escrever uma mensagen, limpar conversar, apresentar Mensagem Embebed, etc).

Não se esqueça de informar o seu email usando o comando /donoservidor; bem como manter o nome do servidor atualizado caso você o modifique.



Comandos relacionados aos Cargos


Comando Função
/ajuda Apresenta uma orientação sobre o Bot Gerente
/listarcargos Apresenta os cargos registrados no servidor
/cargoprovisorio Define o cargo padrão que todo usuário ira receber provisóriamente assim que entrarem no Servidor
O CARGO deve ser informado pelo seu ID
/cargodefinitivo Define o cargo definitivo que os usuários receberao ao serem "Verificados"
O CARGO deve ser informado pelo seu ID
/cargosuporte Define o cargo de Suporte Técnico para usuários que poderão configurar o CachaBOT no servidor.
O CARGO deve ser informado pelo seu ID



Comandos relacionados a Boas Vindas


Comando Função
/salaboasvindas Define a Sala Inicial que todo usuário terá acesso (Boas Vindas)
A SALA deve ser informada pelo seu ID
/msgtitulo Define o TITULO da Mensagem de Boas Vindas
/msgcorpo Define o CONTEUDO da Mensagem de Boas Vindas
Veja como definir o conteúdo abaixo!
/msgcor Define a COR da barra lateral da Mensagem de Boas Vindas
As cores aceitas sao: VERDE, AMARELA, AZUL, LARANJA e BRANCA (epecificar em maiusculas).
A cor DEFAULT e a BRANCA.
/msglinha Define a URL de uma imagem de uma barra (ou linha) que sera apresentada no final da Mensagem de Boas Vindas.
/msgicone Define a URL de uma imagem para ser o ICONE da Mensagem de Boas Vindas.
/boasvindas Apresenta a Mensagem de boas vindas no canal de verificacao.
/msgparticipacao Define a mensagem que o usuário irá receber após clicar em "Participar" (botão da mensagem de boas vindas).



Comandos de uso geral


Comando Função
/clear Limpa todas as mensagens postadas em um canal.
O usuário para executar esse comando deve possuir permissão de Gerenciar Mensagens no canal.
/config Apresenta todas as configurações do servidor para conferência.
Somente um usuário com o Cargo de Suporte Técnico pode ver as configurações
/nomeservidor Atualiza o Nome do Servidor na Base de Dados caso o proprietário o tenha alterado.
Qualquer usuário pode executar esse comando, mas o nome do servidor é obtido do proprio Servidor.
/donoservidor Cadastra na Base de Dados o nome e email do Dono do Servidor.
Somente o proprietário pode executar esse comando.

Como montar sua mensagem de Boas Vindas.


O CachaBOT formata sua mensagem de boas vindas através do comando de barra! Como este meio é muito limitado para formatar uma mensagem, criamos um pequeno codigo de formataçao:
- para mudar de linha utilize o caracter '|' duas vezes sem espaço.
- os codigos de formatacao do Discord podem ser usados
Assim, se no prompt de comando for digitado:
# Seja bem vindo à nossa Comunidade! || || Este espaço tem por objetivo a integração com os clientes, bem como disponibilizar as informações pertinentes a cada cliente dos serviços prestados. Pretendemos assim, ter um registro de todas as informações importantes dos serviços prestados em um único local e de acesso fácil aos clientes e prestador de serviços. || || Esta ferramenta também proporciona: ||* atendimento online||* suporte técnico através de chat, voz e vídeo||* reuniões on-line|| ||Esperamos contribuir assim para um melhor aproveitamento e satisfação do cliente.|| ||*Conte sempre conosco* !
Teremos um resultado igual ao da imagem acima do exemplo de mensagem de boas vindas.


Procedimentos para EXCLUSÃO do CachaBot.


Quando decidir não mais usar o CachaBOT no seu servidor, siga este passo-a-passo:
Primeiro, no Discord, retire o CachaBOT do seu servidor, clicando nele com o botão direito do mouse e no sub-menu que aparecerá, escolha "expulsar CachaBot". Infome o motivo se desejar e clique no botão "Expulsar".
Caso tenha acesso ao Painel Administrativo do CachaBot, faça o login com sua conta e primeiro, delete o seu servidor da Lista de Servidores, depois, caso desejar, delete seu usuário da Lista de Usuários. Pronto, já pode fazer o Logout e o processo está terminado.
Caso NÃO tenha acesso ao Painel Administrativo do CachaBot, basta nos enviar uma mensagem através do menu "Fale Conosco", informando o nome do seu servidor, o email que cadastrou como "Dono do Servidor"(comando de barra do Bot) e o seu Nome (no Discord) que breve iremos retirar todas as suas informações do sistema.
